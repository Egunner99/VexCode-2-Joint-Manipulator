# VexCode-2-Joint-Manipulator

This project contains the control code for a two‑joint VEX robotic arm. The arm uses a distance sensor to detect an object, calculates the joint angles it needs to reach it, and runs a simple dump routine with a servo claw.

Completed for Autonmous Robotics class. 

Developed by Giovanni Mayorga, Andrew Berman, Nino Kramer
